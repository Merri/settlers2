## BlueByte demos and updates

The update and demo files originate from BlueByte's FTP server before it was taken down; the files have been converted
from self-extracting executables to ZIP for easier access on modern devices.

A curiosity of these update and demo archives is that it is possible to collect almost the entire full game out of them.
However it is often just easier to simply get the game from <Link href="https://www.gog.com">GOG.com</Link> as it is
often available for only 2.50 €, you get a pre-configured DOSBox, and there is no digital rights management.

However these files here do have their historical value. Feel free to investigate and research! Also, if you do have the
original installation media and want to experience things "just like back in the day" then these files can be actually
useful.

### Installation instructions

I recommend first to make a backup copy of the whole `DATA` folder just in case something goes wrong. You can find this folder in `C:\BLUEBYTE\SETTLER2\`, `C:\GOG Games\Settlers 2 GOLD\`, `C:\Program Files\GOG.com\Settlers 2 GOLD\`, or whichever path the game has been installed to.

Once you have made a copy of the `DATA` folder you can extract the contents of the downloaded archive. Execute the download, browse to the `DATA` folder and extract files there.

### Rusty Polar

Greenland has beaches and desert.<br />Wasteland is an underground cave full of lava.<br />Winter World is all about snow.

Rusty Polar replaces all of these with new sets!

#### Installation

1. Have a The Settlers II: Gold Edition (or Return to the Roots) game directory ready
2. Clone it if you want to have installation with the original textures
3. Extract zip contents to the game root directory
